  2. Descripcion

    3. Discusión de conceptos de diseño y desarrollo de páginas Web. Aplicación de los lenguajes HTML y JavaScript conjuntamente con CSS para la creación de páginas Web. Creación y adaptación de elementos de multimedios para ser utilizados en páginas interactivas. Requiere 45 horas de conferencia-laboratorio. 3 créditos

  3. Objetivos
    4. Se espera que al finalizar el curso, el estudiante pueda :
    1. Discutir conceptos de disenos y desarrollo de paginas web
    2. Aplicar los lenguajes HTML y JavaScript conjuntamente con CSS para la creación de páginas Web.
    3. Crear y adaptar elementos multimedios para ser utilizadas en páginas interactivas.
    4. Entender las implicaciones éticas, sociales y legales inherentes a la disciplina.
  4. Contenido Tematico
    1. conocer los editores del lenguaje HTML
      1. Historia de los editores de HTML
      2. Introduccion a HTML
        1. Listas enumeradas, no enumeradas y de definicion
        2. Enlaces Internos y esternos
        3. Incorporacion de tablas, imagenes y videos
        4. Diseño de formularios e incorporacion de "meta elements"
    2. Creacion de paginas interactivas.
      1. Creación de páginas interactivas utilizando un editor de HTML.
      2. Publicación de páginas interactivas.
    3. Introduccion a CSS.
      1. Uso de CSS.
      2. Reglas de CSS.
      3. Incorporacion de CSS a una pagina interactiva.
    4. Introduccion a javaScript.
      1. Definicion.
      2. Incorporación devun javascipt a páginas interactivas.
      3. Integración de elementos visuales que utilizan animación vectorial
